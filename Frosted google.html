What is Frosted Google™

Frosted google is a customized css file for google.com that make the boring static white glassormorphism also known as frosted glass or backdrop blur. created by tromoSM.



How to install Frosted Google™

  1. Open Firefox and go to about:config
  2. Search for this preference: toolkit.legacyUserProfileCustomizations.stylesheets
  3. Double-click it to change its value to true
  4. Now Firefox will allow you to use userContent.css and userChrome.css
  5. Close and reopen firefox.

  6. Go to about:support
  7. Scroll to Profile Folder
  8. Click “Open Folder”

    9. Inside that folder, if you don’t see a folder called chrome, create one.

  9. Open the chrome folder inside your profile
  10. Download the .css file from our site
  11. Move the userContent.css file to inside the chrome folder you just created.

  12. Open Firefox and go to about:config
  13. Search for this preference: backdrop-filter
  14. Set layout.css.backdrop-filter.enabled and layout.css.backdrop-filter.force-enabled 'True'
  15. Close and reopen firefox.

  16. Go to https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/ and install tampermonkey.(custom google logo)
  17. Go to github.com/tromoSM/Frosted-google/raw/refs/heads/main/logo.user.js and you will redirected to 'custom logo' installation page.
  18. Click install and you're ready to go.
